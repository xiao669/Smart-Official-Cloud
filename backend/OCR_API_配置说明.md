# 百度OCR API 配置说明

## 📋 申请步骤

### 1. 注册百度智能云账号

访问：https://cloud.baidu.com/product/ocr

1. 点击"立即使用"
2. 注册/登录百度账号
3. 完成实名认证（需要）

### 2. 创建OCR应用

1. 进入"管理控制台"
2. 选择"产品服务" → "人工智能" → "文字识别OCR"
3. 点击"创建应用"
4. 填写应用信息：
   - 应用名称：药品管理系统OCR
   - 接口选择：通用文字识别（标准版）
   - 文字识别包类型：按量后付费
5. 创建成功后，获取：
   - **API Key**
   - **Secret Key**

### 3. 配置到系统

编辑文件：`backend/app/services/ocr.py`

找到第11-13行：

```python
self.api_key = "YOUR_API_KEY"  # 替换为实际的API Key
self.secret_key = "YOUR_SECRET_KEY"  # 替换为实际的Secret Key
```

替换为你的密钥：

```python
self.api_key = "你的API_KEY"  # 例如：xxxxxxxxxxxxxxxxxxx
self.secret_key = "你的SECRET_KEY"  # 例如：yyyyyyyyyyyyyyyyyyyy
```

### 4. 重启后端服务

```bash
# 停止当前服务（Ctrl+C）
# 重新启动
cd backend
python run.py
```

## 🔍 识别能力

### 支持识别的信息
1. ✅ 药品名称
2. ✅ 规格（如：0.25g×24粒）
3. ✅ 生产企业
4. ✅ 批准文号（国药准字）
5. ✅ 生产日期
6. ✅ 有效期至
7. ✅ 批号

### 识别准确率
- 印刷体文字：95%+
- 清晰照片：90%+
- 模糊照片：70%+
- 手写文字：较低

## 📸 拍照建议

为了提高识别准确率：

1. ✅ **光线充足** - 避免阴影和反光
2. ✅ **对焦清晰** - 文字清晰可辨
3. ✅ **拍摄正面** - 包装盒正面或侧面
4. ✅ **保持水平** - 减少倾斜角度
5. ✅ **避免遮挡** - 不要遮挡关键信息
6. ✅ **距离适中** - 不要太近或太远

## ⚠️ 注意事项

### 安全性
- ❌ 不要将API密钥提交到Git仓库
- ✅ 建议使用环境变量存储密钥
- ✅ 定期更换密钥

### 隐私保护
- 图片仅用于识别，不存储
- 识别结果不包含敏感信息
- 符合数据安全规范

### 网络要求
- 需要稳定的网络连接
- 图片上传需要时间（1-3秒）
- 建议WiFi环境下使用

## 🐛 故障排查

### 问题1：识别失败
**可能原因：**
- API密钥配置错误
- 网络连接问题
- 图片格式不支持

**解决方法：**
1. 检查API密钥是否正确
2. 查看后端日志错误信息
3. 确认网络连接正常
4. 尝试重新拍照

### 问题2：识别不准确
**可能原因：**
- 图片模糊
- 光线不足
- 文字被遮挡

**解决方法：**
1. 重新拍照，提高清晰度
2. 调整光线和角度
3. 手动修正识别结果

### 问题3：超出免费额度
**解决方法：**
1. 查看百度云控制台用量统计
2. 考虑升级付费套餐
3. 或限制每日使用次数
