## 功能概述

用户管理功能提供了完整的用户生命周期管理，包括创建、编辑、删除、启用/禁用用户等操作。仅管理员（user_type='admin'）可访问此功能。

## 主要功能

### 1. 用户列表
- 分页显示所有用户
- 显示用户名、真实姓名、用户类型、部门、手机号、邮箱、状态、最后登录时间
- 支持搜索（用户名、真实姓名、部门）
- 支持按用户类型筛选（管理员/经理/操作员）
- 支持批量选择和批量操作

### 2. 创建用户
- 必填字段：用户名、密码、用户类型
- 可选字段：真实姓名、部门、手机号、邮箱
- 数据验证：
  - 用户名：3-50字符，只能包含字母、数字、下划线和连字符
  - 密码：至少6个字符
  - 手机号：11位数字
  - 邮箱：标准邮箱格式
  - 用户类型：admin/manager/operator

### 3. 编辑用户
- 可修改：真实姓名、用户类型、部门、手机号、邮箱、密码（可选）
- 不可修改：用户名
- 相同的数据验证规则

### 4. 删除用户
- 管理员可删除其他用户
- 不能删除自己的账户
- 删除前需要确认

### 5. 重置密码
- 管理员可重置任何用户的密码
- 需要输入新密码并确认
- 密码至少6个字符

### 6. 启用/禁用用户
- 快速切换用户状态
- 不能禁用自己的账户
- 禁用后用户无法登录

### 7. 批量操作
- 支持批量启用/禁用用户
- 自动排除当前登录用户（禁用操作时）
- 显示操作结果统计

## 技术实现

### 后端 API

**基础路径**: `/api/users`

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/users` | 获取用户列表（分页、搜索、筛选） |
| GET | `/api/users/{id}` | 获取用户详情 |
| POST | `/api/users` | 创建新用户 |
| PUT | `/api/users/{id}` | 更新用户信息 |
| DELETE | `/api/users/{id}` | 删除用户 |
| POST | `/api/users/{id}/reset-password` | 重置密码 |
| PATCH | `/api/users/{id}/status` | 切换用户状态 |
| POST | `/api/users/batch-status` | 批量切换状态 |

## 使用说明

### 1. 启动后端服务

```bash
cd backend
python run.py
```

后端服务将在 http://localhost:8000 启动

### 2. 查看 API 文档

访问 http://localhost:8000/docs 查看完整的 API 文档

### 3. 启动前端服务

```bash
cd frontend
npm run dev
```

前端服务将在 http://localhost:5173 启动

### 4. 访问用户管理页面

1. 使用管理员账号登录（admin / admin123）
2. 点击左侧菜单"系统管理" > "用户管理"
3. 或直接访问 http://localhost:5173/system/users

### 5. 运行测试

```bash
cd backend
python test_user_management.py
```

测试脚本将：
- 检查用户表
- 创建测试用户
- 验证搜索功能
- 验证筛选功能
